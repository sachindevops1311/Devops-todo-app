# Devops-todo-app

A collaborative DevOps learning project built by the team.

## Project Description
This is a todo application built to practice DevOps workflows
including Git, GitHub, branching, and pull requests.

## Team Members
- Sachin Meena

## Tech Stack
- Python
- Git & GitHub
- Linux (Ubuntu)

## Setup Instructions
1. Clone the repository
   git clone git@github.com:sachindevops1311/Devops-todo-app.git

2. Navigate to project
   cd Devops-todo-app

3. Create your feature branch
   git checkout -b feature/your-name

## Git Workflow
1. Never push directly to main
2. Always create a feature branch
3. Raise a Pull Request
4. Get it reviewed and merged
